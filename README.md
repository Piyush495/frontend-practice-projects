## 🛠️ How to Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/Piyush495/frontend-practice-projects.git
````



2. Open any project folder, e.g.:

   ```bash
   cd random-emoji-generator
   ```



3. Open `index.html` in your browser.
